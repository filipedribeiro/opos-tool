# OPOS-Abstimmungs-Tool

Automatischer Abgleich von OPOS-Listen (PDF) mit gebuchten Buchungen (Excel).

## Funktionen
- PDF-Parsing der OPOS-Liste
- Excel-Import der gebuchten Buchungen
- Soll/Haben-Spiegelung (Hauptverband ↔ Landesverband)
- Fuzzy-Matching für Rechnungsnummern
- Teilbuchungen werden erkannt und summiert
- Stichtagsabstimmung
- Export der Ergebnisse als CSV
